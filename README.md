📊 Sales Analysis Dashboard (Power BI)
📌 Overview

The Sales Analysis Dashboard is an interactive Power BI project designed to analyze and visualize sales data effectively. It provides a clear overview of business performance by highlighting key metrics such as sales, profit, and trends across different dimensions.

This project demonstrates how raw sales data can be transformed into meaningful insights using data visualization techniques.

🎯 Objectives
To analyze overall sales and profit performance
To identify high-performing categories and regions
To track sales trends over time
To create an interactive and user-friendly dashboard
🗂️ Dataset

The dataset used in this project includes:

Sales data
Profit information
Category and sub-category details
Region and location data
Order date and time

The data has been cleaned and structured to ensure accurate analysis.

📊 Dashboard Description

The Sales Analysis Dashboard presents insights using interactive visualizations such as bar charts, line graphs, pie charts, and KPI cards.

Key highlights include:

Total Sales and Profit overview
Category and sub-category performance
Region-wise sales comparison
Time-based trend analysis

Users can interact with filters and slicers to explore the data dynamically, making the dashboard flexible and easy to use.
